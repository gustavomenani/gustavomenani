### 🚀 O Que Eu Faço

- **Desenvolvimento Full-Stack de Alta Performance:** Construção de aplicações robustas e seguras utilizando C#/.NET no ecossistema de back-end e ecossistemas reativos em TypeScript no front-end.
- **Arquitetura de Dados & Otimização:** Criação de consultas eficientes, modelagem de bancos relacionais (PostgreSQL/SQL Server) com abstração via ORM, estratégias de caching e gerenciamento de filas com Redis.
- **Qualidade, DevOps & Automação:** Automação de infraestrutura em containers com Docker, manipulação programática de mídia (FFmpeg), fluxos inteligentes de IA e cobertura de testes automatizados E2E/Unitários.
